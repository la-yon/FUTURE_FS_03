# Brew & Bloom Cafe - Live Website Project

This project is a professional website demo for a real local cafe business.

## What this website includes

- Modern responsive landing page
- Mobile-friendly navigation
- Service highlights and customer reviews
- Dynamic menu section rendered with JavaScript
- Reservation lead form with confirmation feedback

## Tech stack

- HTML5
- CSS3
- JavaScript

## Run locally

1. Download or clone this project.
2. Open `index.html` in your browser.
